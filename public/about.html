How to install

No coding or developer tools required. First download the add-in manifest, then add it to Excel using whichever method matches how you use Excel.

⬇ Download manifest.prod.xml

Excel on the web · easiest

  1. Open Excel on the web and open or create a workbook.
  2. On the Home tab, click Add-ins → More Add-ins.
  3. Go to My Add-ins → Upload My Add-in.
  4. Choose the manifest.prod.xml you downloaded → Upload.
  5. Click Open APIMaster in the Home ribbon.

Excel desktop — Windows

  1. Open Excel.
  2. Go to Insert → My Add-ins.
  3. Click Upload My Add-in… (not “Manage → XML Expansion Packs”).
  4. Select the manifest.prod.xml file.
  5. Click Open APIMaster in the ribbon.

Excel desktop — Mac

  1. In Finder press ⌘ + ⇧ + G and go to:
    ~/Library/Containers/com.microsoft.Excel/Data/Documents/wef
    Create the folder if it doesn’t exist.
  2. Copy manifest.prod.xml into that folder.
  3. Quit Excel (⌘ + Q) and reopen it.
  4. Go to Insert → My Add-ins → select APIMaster for Excel.
  5. Click Open APIMaster in the ribbon.

Deploying for a whole team? An IT admin can push it to everyone via the Microsoft 365 admin center → Settings → Integrated apps → Upload custom apps, so users don’t install anything themselves.

Enter your key & start

  1. Click Open APIMaster in the Excel Home ribbon to open the sidebar.
  2. Paste your APIMaster API key and click Save & start. The gateway and models are already configured — no other setup needed. Don’t have a key? Get one at apimaster.ai.
  3. Pick a model and ask, e.g. “Explain this workbook” or “Fill B2:B100 with =A2*1.1”.

Privacy

Your API key is stored locally on your device. Spreadsheet content and prompts are sent from your browser directly to the APIMaster gateway and the model you choose. There is no separate application backend collecting your data. See our Privacy Policy and Terms of Use.
